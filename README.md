# 学生宿舍管理系统
 * 本系统采用Springboot+LayUI+Mybatis+MySQL实现了用户的登录，权限管理，数据的增删改查等功能。
* [在线浏览](http://119.3.234.108:8080/)  用户名：root  密码：123 普通用户只有查询功能以及数据导出功能，管理员拥有增删改查，赋予权限的功能。
